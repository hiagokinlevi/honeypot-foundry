# Security Policy

## Responsible Use

This toolkit must only be deployed in environments you own or have explicit written authorization to monitor.

## Reporting Vulnerabilities

To report a vulnerability in this toolkit itself, email **hiagokinlevi@protonmail.com** or open a GitHub security advisory. Do not disclose publicly until a fix is available.

## Operational Safety

- Deploy honeypots in isolated network segments
- Do not connect honeypots to production infrastructure
- Do not store real production credentials in honeypot configurations
- Review all collected telemetry before sharing
- Comply with applicable data protection laws
